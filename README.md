# BigBAT-GUI
GUI for BigBAT. Needs chrome, python, pip and nodejs to be installed.

1. Run `pip install -r requirements.txt`
2. Run `npm install`
3. Run `npm run build`
4. Run `python main.py`

This opens the GUI.
